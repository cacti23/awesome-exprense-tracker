### Expense Tracker App (MERN)

### [Live Site](https://objective-minsky-7d1cd6.netlify.app/)

## Usage

```
change config.env file in config folder
```

```
 npm install
 cd client npm install
 cd ..
 
 # Run front and backend
 npm run dev
 
 # Backend only
 npm run server
 
 # Frontend only
 npm run client
 
 # Build client
 cd client
 npm run build
 
 # Prod
 npm start
```
